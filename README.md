Weather App ☔🌤️

Are you tired of getting caught in the rain? This Android Weather App helps users determine whether it is raining in a specific area by entering latitude and longitude coordinates.

The app uses weather API calls to check current conditions and return a simple result indicating if rain is present or not. Coordinates are entered using geographic latitude and longitude values.

This project was created to strengthen Android development skills and gain hands-on experience with:

API integration and handling API keys
State management in Android development
Error handling and response validation
Working with user input and coordinate-based data
Building Android UI with weather-focused functionality

The main goal of this project was learning how to integrate external APIs into an Android application while improving application state handling and reliability.
API: https://openweathermap.org/api/one-call-3?collection=one_call_api_3.0
